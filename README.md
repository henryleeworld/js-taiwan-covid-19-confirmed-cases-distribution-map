# JS 台灣武漢肺炎（新型冠狀病毒）確診病例分佈圖

以鄉鎮市區為界線，顯示各行政區名稱以及確診病例數字，括號中則是每萬人口確診率，並依據確診病例多寡呈現不同深淺顏色，從中看出不同縣市行政區的確診病例數字以及比例。

## 使用方式
- 把整個專案複製一份到你的電腦裡，這裡指的「內容」不是只有檔案，而是指所有整個專案的歷史紀錄、分支、標籤等內容都會複製一份下來。
```sh
$ git clone
```
